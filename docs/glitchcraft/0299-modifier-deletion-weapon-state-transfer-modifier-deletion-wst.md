# Modifier Deletion Weapon State Transfer `MDWST`
`1.0.0`
Credit is almost impossible to pinpoint. Many people have found this, but most of the research was done by Zas

## Overview
---
Equipping and dropping items in a specific order causes wst with a sideeffect of deleting modifiers.

_?_

## Instructions
---
There are a couple of variants:

Variant 1:
Transfering from a donor to a target 
1. equip an item, it can't be the donor item
2. go to a wall
3. hold L to open the ability menu
4. select the map rune and open the weapon/bow/shield quickmenu
5. equip the donor weapon in the quickmenu without closing it
6. drop it
7. equip the target 
8. let to go to the dpad, the map should open
9. press + and drop the targte
10. unpause

Variant 2:
With this variant the target item IS the donor
1. equip any item
2. go to a wall
3. equip the target/donor item
4. drop it
The effect is that the modifier of the item got deleted.

## Notes
---
Variant 2 can be used to infinitely re-use items on shields that break on impact. The main use for this are infinite bomb jumps, but it can also be used to break pots infinitely.
