# Cold Fuse

## Overview
Interrupting a fuse causes null fe without destroying the connection to the weapon/shield.

## Instructions
```
Overload Cold Fuse
Shock Cold Fuse
Cull Cold Fuse
```

## Notes
Auto Fe:
Fusing a cold fused object to its base causes fuse entanglement
(aka closing the cold fe loop)
Discord
Discord

Additionally, purgatorifying cold fused equippment will tie them to their parent

Pseudo Fuse:
Cold fusing a fuse entangled object causes it to be visually fused in the overworld, but not in the menu

Shield packing:
Combining pseudo fuse with purgatory items allows you to store multiple objects in a single weapon or shield and spawn them into the overworld by using detenglement.
Discord (mulberry)
