# Null Dropping
Aergyl - 16 March 2024

## Overview
Displays a glitched and empty chest quickmenu.

`1.0.0`

## Instructions
1. Empty your inventory in front of a chest, containing the same type as you are dropping
2. unpause and press A on the first frame
