# Camera Pose Glitch

## Overview
Link keeps walking while frozen in the camera animation.

## How To
```
1. Activate the camera rune
2.  Hold ZL
3.  Move in any direction
4.  Press B (Keep holding ZL and Left Stick in any direction)
```
