# Quick Drop Smuggle `QDS`

## Overview
Dropping and equipping weapons in a specific way causes drop smuggle.

`1.0.0`

## Instructions
1. Face links back against a wall
2. pause
3. drop equipped item
4. swap to target
5. drop target
6. pause buffer
7. drop equipped item
8. swap equipment
9. unequip
10. hold up and pause buffer
11. drop equipped item
12. swap equipment
13. unequip
14. unpause
15. pickup dropped equipment
to turn it into a full zuggle, swap equipment and faildrop
